## 📝 Project Overview
This repository contains a simple **blog webpage** created for the **"THE DIGITAL VILLAGE: WEB DESIGN COMPETITION"** held at **Bapu Primary-High School**.

The project highlights the culture, history, food, festivals, and important cities of **Bihar**, one of the historically significant states in India. The blog introduces readers to Bihar’s rich heritage, famous places like **Nalanda University** and **Mahabodhi Temple**, and traditional foods like **Litti Chokha**.

---

## 💻 Technical Skills Demonstrated

### HTML
- Structured webpage using basic HTML elements.
- Use of headings, paragraphs, images, and links.
- Organized blog layout using a container (`div.blog-box`).

### CSS
- **Internal CSS styling** used inside the `<style>` tag.
- Centered blog layout with `width: 60%` and `margin: auto`.
- Rounded corners using `border-radius`.
- Styled background colors for better visual appearance.

### Content Writing
- Information about Bihar’s:
  - Culture
  - Festivals
  - Languages
  - Famous cities
  - Traditional food

## 🖼️ Preview & Setup

To run this project:

1. Download or clone this repository.
2. Make sure these files are in the **same folder**:
   - `index.html`
   - `map.png`
   - `litti.jpg`
   - `nalanda.jpg`
3. Open **index.html** in any modern web browser.
